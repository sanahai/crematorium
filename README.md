# 화장장 서류 안내 서비스

서울·경기·충남·충북권 화장장을 중심으로 다음 기능에 맞춰 다시 정리한 정적 웹사이트입니다.

- 지역 필터 + 검색 중심 목록
- 화장 시간, 사용료, 화장 소요시간 표시
- 사망 유형별 필요서류 분기
- 카카오톡 전달용 공유 문구 생성
- 정적 사이트에서도 동작하는 localStorage 관리자

## 이번 개편 핵심

### 1. 목록 페이지
- 지역 범위를 서울특별시, 경기도, 충청남도, 충청북도로 축소
- 카드에 운영시간, 화장 소요시간, 요금 미리보기 추가
- 카드에서 바로 `서류 공유` 버튼 제공

### 2. 상세 페이지
- 사망 유형: 병사 / 외인사 / 개장유골 / 외국인
- 공통 서류 + 유형별 서류를 함께 렌더링
- 체크리스트 UI 제공
- 카카오톡 공유 모달 제공
- 카카오 SDK 키가 없으면 텍스트 복사 fallback 동작

### 3. 관리자 페이지
- 서버 API 의존 제거
- 브라우저 localStorage 저장 방식으로 변경
- JSON 내보내기 기능 추가
- 기본값 복원 기능 추가

## 데이터 반영 방식

1. `admin.html`에서 수정
2. `JSON 내보내기` 클릭
3. 내려받은 파일로 `js/crematoriums.json` 교체
4. 다시 배포

## 카카오 공유 설정

`js/detail.js` 상단의 아래 값을 설정하면 카카오톡 공유창을 직접 열 수 있습니다.

```javascript
const KAKAO_JAVASCRIPT_KEY = 'YOUR_KEY';
```

키가 비어 있으면 텍스트 복사 방식으로 동작합니다.

## 주의

- 일부 시설은 공식 페이지에서 세부 요금표나 서류가 한 번에 정리되어 있지 않아 `시설 문의 후 최종 확인` 문구를 함께 넣었습니다.
